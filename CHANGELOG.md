# Changelog

## [1.0.0] - 2025-11-24

### Added
- Initial release
- AI adaptive timing engine
- Method rotation (F13/F14/F15/Shift keys)
- Burst mode for natural behavior simulation
- Idle detection (only acts when user is inactive)
- Stealth mode (low priority process)
- Anti-forensics (clears PowerShell history)
- CLI with multiple commands
- Configuration management
- Update notifications
- Support for Teams & Messenger status keeping
- Zero admin privileges required

### Commands
- `gd` - Start with default settings
- `gd --silent` - Run in background
- `gd --stop` - Stop all instances
- `gd --status` - Check running status
- `gd --test` - Test mode (1 min interval)
